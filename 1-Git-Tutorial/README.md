# GitHub Tutorial:

1. Initialising a repository:
```
git init
```

2. Check the status of the git repository
```
git status
```

3. Add file to local git repository
```
git add <file_name>
```

4. Tracking the added file by git
```
git commit -m "Message"
```

5. Sending local repo to remote repository
```
git push origin main
```

6. Declaring remote repo
```
git remote add origin <url.git>
```

7. Download / clone remote repo
```
git clone <url.git>
```

8. Add existing files in remote repo to local repo
```
git pull origin main
```

9. Creating a new branch
```
git branch <branch_name>
```

10. List down all the branches
```
git branch
```

11. Switch to a particular branch
```
git checkout <branch_name>
```

12. Delete a branch
```
git delete -d <branch_name>
```

13. Merge a branch
```
git merge <branch_name>
```

14. Get the entire commit history.
```
git log
```

15. Get oneline commit history.
```
git log --oneline
```
